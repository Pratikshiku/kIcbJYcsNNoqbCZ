## Apartment Repair Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/1d465a39c93b4d78bda70661ffa822b9/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User (Resident), Maintenance Staff, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Room Information
6. Maintenance Request
7. Maintenance Records
8. Maintenance Evaluation


**Maintenance Staff Functions:**
1. Login
2. Change Password
3. Personal Information
4. Maintenance Assignment
5. Reassignment Request
6. Leave Information
7. Maintenance Records
8. Maintenance Evaluation



**Administrator Functions:**
1. Login
2. Personal Center
3. Resident Management
4. Room Information Management
5. Maintenance Staff Management
6. Maintenance Category Management
7. Item Information Management
8. Maintenance Request Management
9. Maintenance Assignment Management
10. Reassignment Request Management
11. Leave Information Management
12. Maintenance Records Management
13. Maintenance Evaluation Management
